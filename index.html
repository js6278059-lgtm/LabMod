How to Play

1. Sandbox Mode

LabMod is a hard science sandbox with real physics.

  • Thermodynamics: Heat transfers between elements. Water boils into steam, lava cools into stone.
  • Chemistry: Reactions occur based on proximity and activation energy.
  • Controls: Space pause/play, C clear, Ctrl+Z undo, Ctrl+Y redo, T tap mode
  • Tools: Pencil, Line, Rectangle, Circle, Spray, Fill Area

2. Multiplayer Servers

Click Servers on the main menu to play with real people!

  • Host: Create a server, set max players, add a PIN, and select which mods to use.
  • Join: Enter a server to spawn as a random colored ball.
  • Controls: A/D to move, W/Space to jump.
  • Chat: Use the chat box in the bottom right to communicate.
  • Admin: The server owner can kick players, clear the canvas, and use all drawing tools.

3. Mods & Elements

Create custom elements with real scientific properties.

  • Thermodynamics: Melting/Boiling points, Specific Heat, Thermal Conductivity
  • Chemistry: Toxicity, Corrosivity, pH, Electrical Conductivity
  • Reactions: Define triggers, products, and effects
  • Visual Scripting: Drag-and-drop blocks to code behaviors
  • Frame Animation: Create multi-frame animated elements

4. Saves & Settings

Backup your progress and customize your experience.

  • Saves: Export/import entire game states
  • Themes: Dark, Light, Cyberpunk, Gameboy, NES
  • Simulation: Toggle temperature, adjust gravity and speed
